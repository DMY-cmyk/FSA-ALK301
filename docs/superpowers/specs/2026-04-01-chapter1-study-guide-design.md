graduate (S2) students who need both conceptual depth and practical formula/ratio fluency.
The guide opens with the big picture (concept map) before drilling into each topic, uses
Colgate Palmolive as the running applied illustration, and ends with curated practice
questions and a quick-reference cheat sheet.

#### 4e. Valuation Models

**Debt Valuation (Valuasi Obligasi)**
```
B_t = I_{t+1}/(1+r)^1 + I_{t+2}/(1+r)^2 + ... + I_{t+n}/(1+r)^n + F/(1+r)^n
```
Where I = interest payment, F = face value, r = yield to maturity

**Equity Valuation Models — Three Approaches:**

1. Dividend Discount Model (DDM):
```
V_t = E(D_{t+1})/(1+k)^1 + E(D_{t+2})/(1+k)^2 + ...
```

2. Free Cash Flow to Equity (FCFE):
```
V_t = E(FCFE_{t+1})/(1+k)^1 + E(FCFE_{t+2})/(1+k)^2 + ...
FCFE = Operating CF − Capital Expenditures + Net Debt Increases
```

3. Residual Income Model (RI):
```
V_t = BV_t + E(RI_{t+1})/(1+k)^1 + E(RI_{t+2})/(1+k)^2 + ...
RI_t = NI_t − (k × BV_{t-1})
```

- Continuing/Terminal Value concept explained
- **Worked example box** (Illustration 1.6 — Labrador/Pitbull): all three models
  yield intrinsic value = $19.31 per share (cost of capital 10%)
